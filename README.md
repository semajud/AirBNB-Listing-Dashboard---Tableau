# AirBNB-Listing-Dashboard---Tableau
Summary: 
This project analyzes 3,800+ Airbnb listings across Seattle using data from 2016. Using Tableau, I built an interactive dashboard with a geographic price map, bedroom and zipcode pricing breakdowns, and a revenue trend line tracking 77% growth from October 2015 to January 2017.

Business Problem:
Understanding how short-term rental pricing and availability vary across a city is valuable for hosts setting competitive rates, investors evaluating neighborhoods, and platforms optimizing supply. This project analyzes a real-world Airbnb dataset to surface pricing patterns, inventory concentration, and revenue trends across Seattle's zip codes.

Methodology:
1. Loaded and explored three relational datasets 
2. Cleaned and joined the data in Tableau
3. Built a geographic price map to visualize average nightly rates by zipcode across Seattle
4. Created a bar chart showing the average price by bedroom count, revealing consistent price scaling from 1 to 6 bedrooms
5. Designed a revenue trend line using calendar data to track weekly projected earnings from October 2015 through January 2017

Skills:
1. Tableau (calculated fields, joins, maps, dashboard design)
2. Data cleaning and transformation

Results and Recommendations:
1. Zipcode 98134 has the highest average nightly rate at $207, nearly double the lowest-performing zipcodes, highlighting strong location-based pricing disparity
2. 1-bedroom listings dominate the Seattle market at 63% of total
3. The summer months show a noticeable plateau in revenue growth
